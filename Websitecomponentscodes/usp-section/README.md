# USP Section

Horizontal row of unique selling proposition cards with icons.
Typically placed below the hero or above the footer.

## Props
- `items`: UspItem[] — Array of USP entries
- `columns?`: 3 | 4 — Layout columns (default: 3)

## UspItem Shape
- `icon`: string — Emoji or icon name
- `title`: string
- `description`: string
