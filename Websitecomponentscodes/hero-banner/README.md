# Hero Banner

A full-width hero section with background image/gradient, headline, subheadline, and CTA button.
Supports dark and light themes. Responsive on mobile.

## Props
- `headline`: string — Main heading text
- `subheadline`: string — Secondary text below headline
- `ctaText`: string — Call-to-action button label
- `ctaHref`: string — CTA link destination
- `backgroundImage?`: string — Optional background image URL
- `theme?`: 'dark' | 'light' — Color scheme (default: dark)
